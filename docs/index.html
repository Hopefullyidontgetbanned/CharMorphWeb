Welcome to Charmorph’s documentation!¶

Charmorph is a Blender Add-on for Character Creation that is explicitly designed to be modular and support any morphology such as Humanoid, Animal, insect, etc.

Check out the Introduction section for further information, including add-on Installation

Note

This Doccumentation is under active development.
